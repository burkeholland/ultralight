Claude Sonnet Orchestrator

Receives your request, analyzes context, and delegates to specialized agents. Integrates results and validates the final output. The brain of the operation.

Claude Opus 4.6 Planner

Researches the codebase, consults documentation, and creates comprehensive implementation plans. Identifies edge cases before a single line of code is written.

GPT-5.3-Codex Coder

Writes production code following strict principles — structure, architecture, naming, error handling. Built for precision and consistency.

Gemini 3.1 Designer

Handles all UI/UX and design work with emphasis on usability, accessibility, and aesthetics. Makes things look and feel right.
